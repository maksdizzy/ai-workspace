## Connect NotebookLM

NotebookLM needs a one-time Google sign-in. The setup flow adapts to your environment automatically.

### GitHub Codespaces

1. Click **Connect NotebookLM** — a virtual desktop opens in a new browser tab
2. Enter password: **workspace**
3. Click **Start Login** — Chromium appears in the virtual desktop
4. Sign in with Google, then switch back to the terminal and press **Enter** to confirm
5. Close the desktop tab — done!

### Local (Docker Desktop)

1. Click **Connect NotebookLM** — a setup command is copied to your clipboard
2. Open a terminal on your local machine and paste the command
3. Sign in with Google in the browser that opens
4. Come back to VS Code and paste your credentials when prompted

### After login

- Create notebooks for each client or project
- Import documents, URLs, and other sources
- Run deep research with automated source gathering
- Generate reports, presentations, and audio overviews
