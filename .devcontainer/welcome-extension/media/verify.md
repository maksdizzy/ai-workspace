## Verify Connection

Click **Verify Connection** to check that NotebookLM is working.

### If it works

You'll see a list of your NotebookLM notebooks (or an empty list if you haven't created any yet).

### If it doesn't

- Make sure you completed the Google sign-in step
- Try the login step again — click **Connect NotebookLM**
- If the desktop viewer doesn't open, check that port 6080 is forwarded (Ports tab)
