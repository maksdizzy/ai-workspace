# AI Workspace

A pre-configured workspace with AI tools, research capabilities, and integrations — ready to use out of the box.

## What's Included

- **Claude Code** — AI assistant in the right panel. Ask it anything, create documents, manage files.
- **NotebookLM** — Google's research tool for deep analysis. Upload documents, get AI-powered summaries and insights.
- **Integrations** — 200+ tool connections (Google Drive, Slack, Gmail, etc.) via the Integrations sidebar.
- **Skills** — Extend Claude's capabilities with community skills from the Skills marketplace.

## Quick Start

Open Claude Code in the right panel and try these prompts:

```
Create a project folder structure for my new project
```

```
Research [topic] and create a summary document
```

```
Help me write a professional email about [subject]
```

```
Show me what tools and integrations are available
```

## Connect Your Tools

### NotebookLM (one-time setup)

1. Open Command Palette (`Ctrl+Shift+P` / `Cmd+Shift+P`)
2. Run **AI Workspace: Connect NotebookLM**
3. Sign in to Google in the browser tab that opens (password: `workspace`)
4. Close the browser tab when done

### Integrations

1. Click the Integrations icon in the left sidebar
2. Connect the tools you use (Google Drive, Slack, Gmail, etc.)

### Skills

1. Click the Skills icon in the left sidebar
2. Browse and install skills to extend Claude's capabilities
